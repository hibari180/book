# Project description
    这个项目出发于自己参加过的创业课程，有少许的商业性质，原本计划为小程序实现功能，在后期的学习中根据自己的开发兴趣和希望的发展方向修改为前后端分离的springboot+vue项目，主要实现后端。
    项目简介：重视社交功能的阅读社区，拥有阅读和交友功能。
    希望实现功能：
    1. 交流社区；
    2. 书籍阅读；
    3. 打卡任务；
    4. 分享到社交软件；
    5. 书评首页推送；
    其他功能在计划书里有详细解释，此处暂不开源。
